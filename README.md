# SPI2

## Notificaciones (Google Chat)
- Define `GCHAT_WEBHOOK_URL` (o `GCHAT_WEBHOOK`) en `backend/.env` con el webhook de tu espacio de Google Chat.
- Opcional: `GCHAT_THREAD_KEY` para agrupar mensajes en el mismo hilo.
- Todas las notificaciones automáticas se envían únicamente por Google Chat (ya no se usa Gmail).
